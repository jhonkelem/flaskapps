Elimination Of Redundant Data In Cloud With Secured Access Control

  • Developed using JSP, Servlets, Javascript, Css
  • The overall aim of the project is developing a system that avoid or eliminate redundant or multiple copies of same encrypted data.
  • aim to solve the issues in deduplication that are being faced by data holders by providing privacy for accessing the file.
  • I combine cloud data de-duplication with data access control
  • I used Symmetric-key algorithms are those which use the same cryptographic keys for both encryption of plaintext and decryption of cipher text.
  • This method also reduces the amount of data to be transferred over a network as the number of bytes are reduced by avoiding duplication.
